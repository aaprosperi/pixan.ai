# Pixan.ai

Plataforma de inteligencia artificial desarrollada con Next.js y desplegada en Vercel.

## Tecnologías

- Next.js
- React
- Tailwind CSS
- Vercel

## Desarrollo

```bash
npm install
npm run dev
```

Abrir [http://localhost:3000](http://localhost:3000) en el navegador.

## Despliegue

Desplegado automáticamente en Vercel mediante GitHub.
